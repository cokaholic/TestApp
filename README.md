# TestApp

## About
WEB+DB PRESS Vol.104 に掲載されている「いきなりiPhoneアプリ開発 -カメラの写真を加工してTwitter投稿！-」特集内の「第2章：アプリ開発をはじめよう」におけるサンプルコードです。

## Features
- 回転させても常に画面中央に表示されるボタン
- ボタンを押したら別の画面に遷移

## Getting Started
cloneしたら、`TestApp.xcodeproj`をXcodeで開いて使用します。

## Requirements
- Xcode9.3+
- Swift4.1
